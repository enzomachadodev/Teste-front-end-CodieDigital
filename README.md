

# Next.js Clean Arch

The project consists of a front-end application with Next.js and TypeScript, consuming the Poke API and an internal Next.js API to obtain information and allow you to schedule medical appointments for your Pok√©mon. The focus of the project was to use the Clean Architecture standard to guarantee modular, scalable and easy to maintain code.

## Requirements

To run the project locally, you need to have the following tools installed:

- Node.js version 18 or higher

## Running locally

Clone the project

```bash
  git clone https://github.com/enzomachadodev/nextjs-clean-arch
```

Enter the project directory

```bash
  cd nextjs-clean-arch
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
